# Product Documentation

## Getting Started

Welcome to our product! Here's how to get started:

1. Unbox your product
2. Connect to power
3. Follow the setup wizard

## Features

- **Smart Mode**: Automatically adjusts settings
- **Energy Saving**: Uses 50% less power
- **Remote Control**: Control via mobile app

## Troubleshooting

### Product won't turn on
- Check power connection
- Try a different outlet
- Contact support if issue persists

### Connectivity issues
- Restart your router
- Ensure product is within range
- Update firmware to latest version
